关于.gitignore

常见场景

  • 忽略操作系统生成的临时文件（如 .DS_Store、Thumbs.db）
  • 忽略 IDE 或编辑器生成的配置文件（如 .vscode、*.suo）
  • 忽略包含敏感信息的文件（如 .env、credentials.json）
  • 忽略自动生成的构建文件（如 node_modules、dist）
